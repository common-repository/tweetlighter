=== tweetlighter - Retweet Outside the Box ===


Retweet outside the box with tweetlighter.  Just highlight the text that you want to tweet and tweetlighter does all of the work for you.  Share the internet without all of those ugly share buttons! `/tags/tweetlighterscreenshot1.png` 

Tags: twitter, retweet, social, sharing, widget

== Installation == 
1. Activate tweetlighter through the 'Plugins' menu in WordPress.
2.  Place `<?php if(function_exists('wp_select')) { wp_select(); } ?>` code in footer.php of selected theme.
